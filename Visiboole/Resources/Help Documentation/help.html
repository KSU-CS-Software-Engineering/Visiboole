Following is an introduction on how to use VisiBoole’s features. This document is purely instructional: it contains screenshots and descriptions of the user interface and a walkthrough of VisiBoole. The full description on how to use the programming language itself is available in the menu under Help -> Syntax Guide

Toolbar:

The toolbar has menus that allow you to create and edit VisiBoole designs, enhance your designing experience, and learn more about what VisiBoole is. The toolbar also has shortcut icons that allow you to quickly perform some common tasks.

File Menu:

New (Ctrl + N) allows you to create a new VisiBoole design.

Open (Ctrl + O) allows you to open an existing VisiBoole design.

Save (Ctrl + S) allows you to save the VisiBoole design that is currently being edited.

Save As allows you to save the VisiBoole design that is currently being edited under a new name.

Close allows you to close the VisiBoole design that is currently being edited.

Close All allows you to close all VisiBoole designs that are currently being edited.

Exit allows you to exit the application.

Edit Menu:

Undo (Ctrl + Z) allows you to undo the most recent change you have made in the current VisiBoole design.

Redo (Ctrl + Y) allows you to redo the most recent change you have made in the current VisiBoole design.

Cut (Ctrl + X) allows you to remove and copy text from the VisiBoole design that is currently being edited.

Copy (Ctrl + C) allows you to copy text from the VisiBoole design that is currently being edited.

Paste (Ctrl + V) allows you paste copied text into the VisiBoole design that is currently being edited.

Select All (Ctrl + A) allows you to select all the text in the VisiBoole design that is currently being edited.

View Menu:

Increase Font (Ctrl & +) allows you to increase the text size of the design editor and simulator.

Decrease Font (Ctrl & -) allows you to decrease the text size of the design editor and simulator.

Light Theme allows you change the theme of the application to a light theme.

Dark Theme allows you change the theme of the application to a dark theme.

Toggle Colorblind Mode allows you to enable/disable colorblind mode for the application.

Toggle Simulator Comments allows you to enable/disable design comments in the design simulator.

Toggle Simulator Semicolons allows you to enable/disable ending semicolons in the design simulator.

Help Menu:

Syntax Guide allows you to view the syntax rules for VisiBoole.

User Guide (Current Document) allows you to learn how to use the VisiBoole application.

VisiBoole Overview allows you to learn more about VisiBoole.

About VisiBoole contains the current version number and licensing/copyright information.

Toolbar Shortcuts:

(From Left to Right)

New (Ctrl + N) allows you to create a new VisiBoole design.

Open (Ctrl + O) allows you to open an existing VisiBoole design.

Save (Ctrl + S) allows you to save the VisiBoole design that is currently being edited.

Save All (Ctrl + Shift + S) allows you to save all the VisiBoole designs that are currently being edited.

Undo (Ctrl + Z) allows you to undo the most recent change you have made in the current VisiBoole design.

Edit (Ctrl + E) allows you to change from simulation (run) mode to editor mode.

Run (Ctrl + R) allows you to change from editor mode to simulation (run) mode.

Run Menu allows you to view the run menu.

Run Menu:

New State (Ctrl + R) allows you to run the VisiBoole design that is currently opened with a new state (no persistent variable values).

Previous State allows you to run the VisiBoole design that is currently opened with previous variable values from the last execution of that design.

Design Browser:

The design browser contains all VisiBoole designs that are currently opened in the application. You can double click designs to select, open and edit them.

Design Editor:

The design editor is where you will edit your VisiBoole designs.

The top of the editor has design tabs for each design that you have currently opened. Clicking on a tab will select and open that design. You can click on the red X on the tab to close that design. Designs that have unsaved changes will have a * before the design name.

The bottom portion of the editor is where your design text goes. On the left side you can view line numbers of your design text.

Design Simulator:

The design simulator is where you will run your VisiBoole designs.

The top of the simulator has buttons for clock ticking in designs. Tick button allows you to tick sequential assignments once for their next value. X Ticks button allows you to tick sequential assignments the specified number of times.

The bottom portion of the simulator is where the output of your design goes. Simulation tabs are used for the current design and any subdesigns of the current design that is being simulated. Note: Only one subdesign of a design can be opened at any given time. In the top level design, you can click on independent variables to toggle their values from off to on or on to off.

Design Simulator Errors:

The design simulator error box will list errors from your design. Each line (starting with the line number of the error) in the error box represents a new error in your design. Note: Only the first error of a line will be listed. If multiple errors exist on a line, you must first fix the first error to view the next error.
