Designs:
The leftmost section under the
heading ‘Designs:’ is the file browser of
VisiBoole. All open files are listed here,
and while in Edit mode double clicking
on a design name will open that file.
Particularly useful if many files are open
at once.
Main Window:
When VisiBoole first opens the main window contains only a prompt to open a
file. Once a file is opened or created the window is filled with the text editor and the
program will be in Edit mode. Closing all files in the editor returns VisiBoole to this
starting screen.
Edit Mode:
Code is typed in text editor, much like the standard notepad application available
on most computers. The name of the file is shown on the Designs tree on the leftmost
part of the window and on the tab of the text editor.
The text editor is tab-based: Multiple files can
be open simultaneously, each can be closed
with the red X icon present on each tab, and if
a file has unsaved changes an asterisk (*) is
placed beside the close icon:
